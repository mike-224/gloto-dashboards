# Diseño de Dashboards
- **Layout CSS Grid**: Usar `display: grid` con `minmax` para tarjetas responsive.
- **Componentes Visuales**:
    - **Tarjetas/Stats**: Bordes semánticos de color (e.g., verde para ingresos, rojo para alertas).
    - **Listas**: Filas compactas con flexbox para densidad de información.
    - **Tabs**: Navegación interna usando data-attributes y JS vainilla ligero.
- **UX Patterns**:
    - Indicadores de tendencia (↑ ↓) con colores.
    - Botones de "Actualizar Análisis" para procesos pesados.
    - Empty States informativos ("Necesitas más datos...").
